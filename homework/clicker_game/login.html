小松模擬器 遊戲介紹

這是一個簡單的點擊抽卡遊戲，靈感來自於「小松模擬器」。你可以點擊畫面上的圖片來抽取不同稀有度的角色，並在左側圖鑑中收集你的成果。

原始版本是使用 Flask（Python 網頁框架）開發，並搭配伺服器端資料庫來儲存玩家的收集紀錄。

目前這個版本為了能夠部署在 GitHub Pages，已經改為純前端 HTML、CSS、JavaScript，所有資料都儲存在瀏覽器的 localStorage，無法連接資料庫或伺服器，也無法跨裝置同步你的收集進度。

你可以在 GitHub 專案頁面 查看本遊戲的原始碼與專案說明，歡迎參考與學習！

如果你重新整理或清除瀏覽器資料，圖鑑會被重置。
若想體驗伺服器版本，請於本地端自行架設 Flask 環境。

※ 本遊戲適合部署於 GitHub Pages 或任何靜態網頁空間。
